# 新手導師養成記

台灣國中**導師班級經營**養成遊戲——純前端、零依賴、開瀏覽器即玩。

從開學到畢業 40 週，每天一張情境卡，用《教師法》、正向管教與通報法規做判斷，
練成你的**導師段位**，並依一整年的選擇走向 **30 種班級結局**（20 正向 + 10 待努力）。

## 玩法

- **40 週校園日常**：每週 5 天，每天一題 4 選 1 情境，正解嚴格綁法源。
- **法規素養分 + 導師段位**：答對累積素養，段位從新手→合格→專業→法規專家。
- **班級五維**：凝聚／班風／信任／榮譽／身教，全程即時可見，決定你的結局。
- **資源**：心力（HP）與班費，要顧好自己也要顧好班級。
- **連對系統**：連續最佳解越多手感越燙，斷掉會心痛。
- **30 種結局**：依結業時的最終狀態自動判定，每種都有專屬插畫。
- **每局不同**：選項隨機洗牌、平時週依種子抽題，重玩體驗不重複。
- **男／女老師**：選誰，整套班級畫面（80 張場景）跟著換。

## 技術

- 純 vanilla JS（ES modules），無建置工具、無框架。
- 直接開 `index.html`（會自動進入 `app.html`）即可遊玩。
- 本機開發：`python3 -m http.server 8765` 後開 `http://localhost:8765/`。
- 設計骨架：Octalysis 八角理論八大核心驅動力。

## 結構

```
index.html      入口（導向 app.html）
app.html        主遊戲（首頁→選性別→遊戲→結局）
endings.html    30 種結局圖鑑
js/state.js     狀態、段位、存檔
js/engine.js    計分、效果、隱藏連對勳章
js/gameData.js  40 週情境與法規題庫
js/endings.js   30 結局判定與達成機制
assets/scenes/  80 張週場景圖（男/女 ×40）
assets/endings/ 30 張結局插畫
assets/refs/    角色卡
```

插畫以 Q 版日式漫畫風格（codex / ChatGPT 生成）製作。
